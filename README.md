# 📄 文件内容比较工具

一款轻量级、跨格式的文本比较工具，帮助您快速找出两个文档之间的文字差异。

## ✨ 主要功能
- 支持格式：**.txt / .docx / .xlsx / .pptx / .pdf**
- **三种比较模式**：严格（全部字符） / 忽略中文空格 / 忽略所有空白
- **输出差异详情**：显示位置（行号/段落/幻灯片/表格）及前后文本
- **图形界面**：无需命令行，双击即用
- **便携免安装**：单个文件夹，可放在U盘或任意位置

## 🚀 适合人群
- 学生、老师、办公人员：检查作业、合同、报告修改
- 程序员：对比配置文件、日志、代码片段
- 任何需要快速核对文本差异的人
